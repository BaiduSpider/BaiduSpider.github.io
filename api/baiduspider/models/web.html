    baiduspider.models.web ⚓︎

    网页搜索返回值模型模块

    此文件定义的所有现有的网页搜索返回值模型并编写了自动构建函数。

    WebBaike ⚓︎

    百科搜索结果模型

    这是一个遵照BaiduSpider网页搜索百科搜索结果结果模型创建的返回模型类。

    Attributes:

    Name Type Description
    cover str

    百科封面图片（视频）链接
    cover_type str

    百科封面类型，图片时为"image"，视频时为"video"
    des str

    百科简介
    title str

    百科标题
    url str

    百科链接
    plain dict

    源搜索结果字典
